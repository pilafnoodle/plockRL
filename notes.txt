pipeline steps:
1. use parse_raw_data.py to convert raw states [lidar,steering,speed] into a sarsd tuple
2. use recompute_rewards.py to calculate different rewards without having to rencode lidar, make sure infile is the same file used to generate the sarsd
3. use td3_train.py to train model
4. use off_policy_inference.py to run 

example usage:

#go from raw states to sarsd buffer
python3 parse_raw_data.py --infile raw_states_golf.csv --outfile sarsd_buffer_iris.csv

#run this while changing reward function to create different buffers
#very important, the raw states file must be the same one the sarsd buffer was parsed from because it grabs parallel lidar scans
#avoid having to recompress scans for the sarsd buffer
python3 recompute_rewards.py --infile raw_states_golf.csv --outfile sarsd_buffer_iris.csv

#now train
python3 td3_train.py --infile sarsd_buffer_iris.csv --name iris --starter [model name]

#run
python3 off_policy_inference.py --infile td3_iris.pth
